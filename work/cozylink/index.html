CozyLink

Building a housing rental app for college students, including a design system and brand illustrations

Period 2023 â€“ 2024
Role UI/UX Designer & Developer

Problem

College students face significant challenges finding suitable housing and compatible roommates. Existing platforms lack user-friendly interfaces and don't effectively match students based on lifestyle preferences and compatibility.

Solution

CozyLink is a comprehensive housing rental app designed specifically for college students, featuring an intuitive interface, advanced matching algorithms, and a complete design system with custom illustrations.

Key Features

  • Customizable avatar system for personalization
  • Advanced roommate matching algorithm
  • Comprehensive property listings with detailed information
  • Integrated messaging system
  • Custom design system and brand illustrations
  • Mobile-first responsive design

Results

CozyLink successfully addresses the housing challenges faced by college students through a user-centered design approach. The app's intuitive interface and comprehensive feature set provide students with an effective platform for finding both housing and compatible roommates.

Learnings

This project taught me the importance of understanding user needs through research and creating comprehensive design systems. Building a complete app from concept to implementation reinforced the value of iterative design and user testing in creating effective digital solutions.
